# Common ComfyUI Model Fields

Fields that often contain model filenames or model-like references:

- `ckpt_name`, `checkpoint`, `checkpoint_name`
- `lora_name`, `loras`, `lora`
- `vae_name`, `vae`
- `clip_name`, `clip`, `clip_l`, `clip_g`
- `unet_name`, `diffusion_model_name`, `model_name`
- `control_net_name`, `controlnet_name`, `control_net`
- `upscale_model`, `upscale_model_name`
- `embedding`, `embeddings`
- `style_model_name`, `gligen_name`, `photomaker_model_name`

Common optimization fields:

- `width`, `height`, `empty_latent_width`, `empty_latent_height`
- `batch_size`, `batch`
- `steps`, `cfg`, `denoise`
- `scale_by`, `upscale_by`, `hires_steps`
